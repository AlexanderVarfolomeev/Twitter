Требования к задаче.
Создать приложение - аналог твиттера.
Реализована авторизация (возможен вход через сервисы vk, gmail)
Главная страница представляет собой ленту твитов людей на которые подписан пользователь.
Также есть страница со всеми твитами. Доступна сортировка, по популярности (за день, неделю, месяц), по дате.
Главная страница пользователя представляет собой краткое описание, фото, и личный блог.
Пользователь может подписаться на другого пользователя.
Посты можно лайкать и комментировать.
К постам можно прикладывать картинки.
